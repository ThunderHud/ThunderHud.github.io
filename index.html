博客搭建

hexo博客搭建

注：以下操作都根据 BiliBili 视频博主codesheep的hexo博客搭建视频进行操作。

  1. 下载node.js 官网下载npm包

node -v #查看是否安装成功，查看版本号

  1. 下载淘宝镜像，加速下载。

npm install -g cnpm –registry=https://registry.npm.taobao.org
cnpm -v #查看是否安装成功，查看版本号

  1. 下载hexo

cnpm hexo install -g hexo-cli
hexo -v #查看hexo框架。

  1. 创建一个空目录，用来作博客文件夹（目录），当操作失误修改无果，直接删掉就行。

mkdir /blog #根据视频演示，创建Blog文件夹，文件夹名可自行修改。
cd /blog #进入blog文件夹

  1. hexo初始化，如果有梯子建议用梯子进行下载，速度会快一点，无梯子则需要等待一会。

hexo init #安装hexo的各种所需要的文件。

#会显示如下：
#INFO Cloning hexo-starter https://github.com/hexojs/hexo-starter.git
#INFO Install dependencies
#INFO Start blogging with Hexo!

[root@docker-lnmp blog]# ls -l

#安装完成

  1. 启动hexo

[root@docker-lnmp blog]# hexo s
INFO Validating config
INFO Start processing
INFO Hexo is running at http://localhost:4000/ . Press Ctrl+C to stop.

#在网页输入：localhost:4000 或 虚拟机Ip：xxx.xxx.xxx.xxx:4000 即可。
